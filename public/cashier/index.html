แต้มสะสม / ของรางวัล

ลูกค้าที่จ่ายเงินแล้วได้แต้มอัตโนมัติ — ผูกเบอร์ที่เคาน์เตอร์ (📱 แต้ม) หรือสั่งผ่านไลน์

สะสมครบ แก้ว = ฟรี 1 แก้ว (ไม่เกิน 49฿)
🎁 สั่งผ่านไลน์ครั้งแรก แถมฟรี ดวง (0 = ปิด)

จ่ายเงินแล้วได้ 1 ดวง/แก้ว — ลูกค้าหน้าร้านผูกเบอร์ที่เคาน์เตอร์ (ปุ่ม 📱 แต้ม) ก่อนรับเงิน · ลูกค้าสั่งผ่านไลน์ได้อัตโนมัติ · โบนัสต้อนรับช่วยให้เริ่มสะสมเร็วขึ้น

ระดับสมาชิก (จากยอดสะสมตลอดชีพ)

ลูกค้าเลื่อนระดับเมื่อสะสมถึงเกณฑ์ · ตั้งส่วนลด % (0 = ไม่ลด) เพื่อให้ส่วนลดสมาชิกอัตโนมัติ · ใส่สิทธิประโยชน์อธิบายสิทธิ์เพิ่มเติม

ของรางวัล

เพิ่มของรางวัลก่อน แล้วกด 📷 ที่รายการเพื่ออัปโหลดรูปจากเครื่องได้

เช็ค / แลกแต้มลูกค้า

วางรหัสลูกค้า LINE (จากหน้าลูกค้า) เพื่อดูแต้มและแลกของรางวัล

ส่งข้อความ LINE

ส่งโปรโมชั่นถึงลูกค้า LINE ทุกคนที่เคยสั่งผ่านร้านคุณ

LINE ยังไม่ได้เชื่อมต่อ — ข้อความจะถูกบันทึกไว้แต่ไม่ถูกส่งจริง ตั้งค่า LINE Channel Token ในหน้าตั้งค่าระบบ
