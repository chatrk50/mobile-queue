เชื่อมต่อ LINE ทีละขั้นตอน

สำหรับแพ็กเกจที่มี LINE (ลูกค้าสั่งผ่านไลน์ + แจ้งคิว + สะสมแต้ม) · ใช้เวลา ~15–20 นาที

ไม่อยากทำเองตอนนี้? เริ่มแบบ POS ขายหน้าร้าน + คิว ได้ทันทีโดยไม่ต้องตั้ง LINE — แล้วค่อยกลับมาเชื่อม LINE ทีหลัง · ลูกค้าแพ็กเกจ Pro/Business มีทีมช่วยตั้งให้ (ดูปุ่มช่วยเหลือด้านล่าง)

สิ่งที่ต้องเตรียม: ① LINE Official Account (บัญชี @ร้าน) และ ② LINE Developers (ที่ออก token/LIFF) — กรอก 3 ค่าเข้าระบบเรา: Channel access token, Channel secret, LIFF ID

1 มี LINE Official Account (OA)

สร้าง/ใช้บัญชีทางการของร้านที่ oa.line.me หรือในแอป LINE (ฟรี) — นี่คือ @ร้าน ที่ลูกค้าจะแอดเป็นเพื่อน

ส่วนใหญ่ร้านมี OA อยู่แล้ว — ข้ามไปข้อ 2 ได้

2 สร้าง Provider ใน LINE Developers

เข้า developers.line.biz/console → ล็อกอินด้วยบัญชี LINE เดียวกับ OA → กด Create a new provider → ตั้งชื่อ (เช่นชื่อร้าน)

3 Messaging API channel → token + secret

ในProvider → Create a Messaging API channel (หรือเชื่อมจาก OA ที่มี) → กรอกชื่อ/หมวด

หา Channel secret ที่แท็บ Basic settings · หา Channel access token ที่แท็บ Messaging API → กด Issue (เลือกแบบ long-lived)

4 ตั้ง Webhook

ที่แท็บ Messaging API → ช่อง Webhook URL วางลิงก์จากหน้า "เชื่อมต่อ LINE" ของเรา (กดปุ่มก๊อปในระบบ) → เปิด Use webhook → กด Verify

รูปแบบลิงก์: https://…/b/<ร้านคุณ>/line/webhook (ระบบก๊อปให้)

5 LINE Login channel → LIFF (สำหรับ "สั่งในไลน์")

ในProvider เดิม → Create a LINE Login channel → เข้าแท็บ LIFF → Add

  • Endpoint URL = ลิงก์หน้า LIFF ของร้าน (ระบบก๊อปให้): https://…/b/<ร้านคุณ>/liff/
  • Size = Full · Scopes = profile + openid
คัดลอก LIFF ID (เช่น 1234567890-abcdEFGh) จากหน้า LIFF
ถ้ายังไม่อยากให้ลูกค้าสั่งในไลน์ ข้อ 5 ข้ามได้ — ใส่แค่ token+secret ก็แจ้งคิว/แต้มทาง LINE ได้แล้ว (ค่อยเพิ่ม LIFF ภายหลัง)

6 วางค่าเข้าระบบ + กดตรวจสอบ

กลับมาที่หน้าแคชเชียร์ → ตั้งค่าระบบ → เชื่อมต่อ LINE → วาง token / secret / LIFF ID → กด ตรวจสอบ ระบบจะเช็คให้ทันทีว่าต่อกับ @ร้านคุณถูกต้องเรียบร้อย
